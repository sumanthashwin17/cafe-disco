Cafe-Disco

A Full-Stack Coffee Management Web Application built using Spring Boot, React.js, and MySQL.  
This application allows admin users to manage coffee items through CRUD operations with a clean and responsive UI.


Project Tech Stack
------------------

Frontend
----------
React.js
React Router
Axios
CSS 

Backend
--------
Java
Spring Boot
Spring Data JPA
MySQL
REST APIs

Tools
------
Git & GitHub
Postman
Spring Tool Suite (STS)

 Features
 ---------
- Add new coffee items
- View all available coffee products
- Update coffee details and pricing
- Delete coffee records
- REST API integration between frontend and backend
- Exception handling and validation
